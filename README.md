## 🌟 Chat With PDF 🌟

AI powered web app to chat with your documents. A RAG application that work as a external knowledge base and provide answer from the given context.

### 🛠️ Tools Used

- Nextjs
- Vercel AI SDK
- Langchain JS
- Pinecone DB
